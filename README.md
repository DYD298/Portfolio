# Developer Portfolio

A high-performance, aesthetically pleasing portfolio website built with HTML, CSS, and JavaScript.

## How to View

Simply open the `index.html` file in your web browser. No installation or build steps required.

## Features
- **Dark Mode Aesthetic**: Modern, sleek design with glassmorphism elements.
- **Responsive**: Works on desktop and mobile.
- **Micro-animations**: Smooth transitions and cursor effects.
- **Vanilla Tech Stack**: Pure HTML/CSS/JS for maximum performance and simplicity.

## Editing
- **Styles**: Edit `style.css` to change colors, fonts, and layout.
- **Content**: Edit `index.html` to update text and images.
- **Logic**: Edit `script.js` for interactive behaviors.
